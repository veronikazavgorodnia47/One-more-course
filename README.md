# Ще один курс?

A small decision-helper tool for designers to quickly evaluate whether a course or event is worth their time.

## What it does

Asks 4–5 questions based on the type of learning opportunity (one-off event vs. course) and your current workload, then gives a clear verdict: **go**, **maybe**, or **skip**.

## How to use

No build step, no dependencies. Just open `index.html` in a browser.

## Stack

- Vanilla HTML, CSS, JavaScript
- No frameworks, no bundler
- Dark mode support with `localStorage` persistence
